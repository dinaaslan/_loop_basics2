# _loop_basics2
this code contains the following functions: 

1. A function that will change all positive numbers to “big”

2. A function that will replace the last item in the list with the number of positive values in the list

3. A function that takes a list and returns the sum of all the values in the list.

4. A function that takes a list and returns the average of all the values.

5. A function that takes a list and returns the length of the list

6. A function that takes a list of numbers and returns the minimum value in the list. If the list is empty, have the function return False.

7. A function that takes a list and returns the maximum value in the list. If the list is. empty, have the function return False.

8. A function that takes a list and returns a dictionary that has the sum Total, average, minimum, maximum and length of the list.

9. A function that takes a list and return that list with values reversed without creating a second list.
